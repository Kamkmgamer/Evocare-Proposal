# Evocare Project Specification

The aim of this project is to design and develop a fully custom-built website for **Evocare** with a bespoke backend system tailored specifically to the automotive detailing and enhancement industry.

By building the platform from scratch, we ensure:

- **Full control** over performance and functionality
- **Zero dependency** on plugins or pre-built frameworks
- **Seamless integration** with advanced configurators
- A **scalable architecture** ready for future expansions
- A **secure and optimized environment** for SEO and marketing

This custom platform will include a quotation automation system, advanced tracking tools, deposit payments, and a tailored admin dashboard for complete business management.

---

## 2. Objectives

- Develop a fully custom-coded luxury website aligned with Evocare’s brand.
- Provide an advanced car configuration experience.
- Automate quotation creation and follow-ups.
- Enable tracking of customer requests and payment status.

## 4. Custom Admin Dashboard Features (Backend)

### 1 — Content Publishing System + Full SEO Tools

A complete article/post publishing module including:

- Create / edit / delete articles.
- Categories & tags system.
- Featured image tools.
- **Full SEO control per article**: Meta title, description, OG tags, Schema, Canonical URL, Slug optimization, and Header tags (H1–H4).
- Auto image optimization.
- Structured blog listing and search.

### 2 — Customer Order Tracking System

A full CRM-style module built specifically for Evocare:

- Track all customer inquiries & car configuration requests.
- **Status Assignment**: New, In Process, Quotation Sent, Awaiting Payment, Completed.
- Internal notes & action logs.
- Ability to send quotations via custom URL.
- Email/WhatsApp delivery with "Viewed" tracking.
- Track deposit / payment status.
- Full reporting dashboard.

### 3 — Full Page SEO Control (For Every Page)

Each page of the website will include:

- Editable meta title & meta description.
- Open Graph controls (Facebook/Instagram previews).
- Structured Schema markup.
- Index/no-index and canonical controls.
- Automatic sitemap generation.
- Real-time SEO score.

## 6. Technical Specifications

- **Custom-coded**: No WordPress, no templates.

- **Fully Responsive**: Mobile, tablet, and desktop.
- **SEO-optimized architecture**.
- **Real-time quotation engine**.
- **Custom-built CRM** for request tracking.
- **Security**: HTTPS, firewall, rate limiting, and brute-force protection.
- **Performance**: Caching, minified assets, and optimized images.
- **Scalability**: Designed for future modules (loyalty, multi-branch, etc.).
